## Run Locally

Clone the project

```bash
  git clone git@github.com:vineet98-code/Admin-ui.git
```

Navigate to the project

```bash
  cd .\Tesla\tesla-clone
```

Install dependencies

```bash
  npm install
```

Start the server



view client in the browser

Local: http://localhost:3000
